## 📖 Description

This project was developed for the **Knowledge-Based Systems (SBC)**
course and focuses on the design and implementation of a rule-based
intelligent system.

The objective is to build a knowledge-based system capable of reasoning
over a structured knowledge representation using inference rules.

The system includes:

-   A knowledge base (facts and rules)\
-   An inference engine\
-   Logical reasoning capabilities\
-   Query handling\
-   Example use cases and validation scenarios

------------------------------------------------------------------------

## 🧠 Academic Objectives

The main goals of this project are:

-   Model domain knowledge formally\
-   Implement rule-based reasoning mechanisms\
-   Separate knowledge representation from inference logic\
-   Analyze system behavior under different knowledge configurations\
-   Validate correctness and consistency of the knowledge base

## 🧩 System Architecture

### 1. Knowledge Base

-   Stores facts and rules\
-   Defines relationships between entities

### 2. Inference Engine

-   Applies reasoning over the knowledge base\
-   Uses forward chaining and/or backward chaining

### 3. User Interaction Layer

-   Allows queries to be submitted\
-   Returns inferred conclusions
